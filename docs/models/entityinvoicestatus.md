# EntityInvoiceStatus

Status of the invoice.

* `open` — The invoice is not paid yet.
* `paid` — The invoice is paid.
* `overdue` — Payment of the invoice is overdue.


## Values

| Name      | Value     |
| --------- | --------- |
| `OPEN`    | open      |
| `PAID`    | paid      |
| `OVERDUE` | overdue   |
