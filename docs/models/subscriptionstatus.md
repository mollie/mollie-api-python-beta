# SubscriptionStatus

The subscription's current status is directly related to the status of the underlying customer or mandate that is
enabling the subscription.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `PENDING`   | pending     |
| `ACTIVE`    | active      |
| `CANCELED`  | canceled    |
| `SUSPENDED` | suspended   |
| `COMPLETED` | completed   |
