# OrderLineCategories

A comma-separated list of the line categories you support in your checkout.

Example: `/v2/methods?orderLineCategories=eco,meal`


## Values

| Name            | Value           |
| --------------- | --------------- |
| `ECO`           | eco             |
| `GIFT`          | gift            |
| `MEAL`          | meal            |
| `SPORT_CULTURE` | sport_culture   |
| `ADDITIONAL`    | additional      |
| `CONSUME`       | consume         |
