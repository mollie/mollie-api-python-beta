# MandateResponseStatus

The status of the mandate. A status can be `pending` for mandates when the first payment is not yet finalized, or
when we did not received the IBAN yet from the first payment.


## Values

| Name      | Value     |
| --------- | --------- |
| `VALID`   | valid     |
| `PENDING` | pending   |
| `INVALID` | invalid   |
