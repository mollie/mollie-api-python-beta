## list

Retrieve a list of all physical point-of-sale devices.

The results are paginated.

> ðŸ”‘ Access with
>
> [API key](/reference/authentication)
>
> [Access token with **terminals.read**](/reference/authentication)

### Example Usage

<!-- UsageSnippet language="python" operationID="list-terminals" method="get" path="/terminals" -->
```python
import mollie
from mollie import Client
import os


with Client(
    security=mollie.Security(
        api_key=os.getenv("CLIENT_API_KEY", ""),
    ),
) as client:

    res = client.terminals.list(from_="term_vytxeTZskVKR7C7WgdSP3d", limit=50, sort="desc", testmode=False)

    # Handle response
    print(res)

```
